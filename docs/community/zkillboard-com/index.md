## About

zKillboard.com is the premier killboard for EVE Online. Track players, corporations, and alliances, analyze battle reports, and monitor PvP anywhere in New Eden.

It aggregates killmails and presents them in a way that helps players quickly answer practical questions:

- Who is active in a system right now?
- What ships and doctrines are being used?
- How has a pilot, corporation, or alliance been performing over time?

If you are planning routes, evaluating risk, or reviewing a fight, zKillboard is often the first stop.

## Key Features

zKillboard includes a broad set of combat intelligence capabilities. In short, it helps with:

- **Tracking entities**: follow activity for players, corporations, alliances, systems, and regions.
- **Studying engagements**: inspect killmails, losses, and battle outcomes with practical filters.
- **Operational awareness**: monitor PvP activity trends to support scouting and planning.
- **Data access and integrations**: consume zKillboard data in other community tools and workflows.

For the full and up to date feature list, see: [zKillboard Features](https://zkillboard.com/information/features/)

## Common Use Cases

- **Scouting and travel planning**: check recent losses and kills in choke points before moving assets.
- **Target research**: inspect an opponent's recent fits, fleet composition, and active time windows.
- **After action review**: validate battle reports, compare fittings, and identify tactical mistakes.
- **Alliance intelligence**: monitor regional activity trends and strategic hotspots.

zKillboard provides API and data endpoints that many EVE tools consume for killmail and intelligence workflows.
