## Description

The purpose of this website is to visually display killmails in a visually appealing format providing the end user the opportunity to find content.

## Features

- Region wide display with visual aid for killmails in the past hour.

- Information tab for a flatlist of region killmails in the past hour.

- Search for any region that might be interesting to you.

- Website application programmer interface [documentation](https://api.astrocartics.xyz/swagger/index.html)
